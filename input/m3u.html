M3U Support¶

M3U is a fairly standardized file format that is used by a variety of software, such as Virtual DJ.

Instructions¶

  1. Open Settings from the Now Playing icon

  2. Select Input Source from left-hand column

  1. Select the M3U from the list of available input sources.

  2. Select M3U from the left-hand column.

  3. Enter or, using the button, select the directory where your m3u files will appear. For Virtual DJ, this location should be your Documents/Virtual DJ/History directory.

  1. Click Save

Now Playing will now read that directory, reading the most current m3u file for the currently playing track information.
