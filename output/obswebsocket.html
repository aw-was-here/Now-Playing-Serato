OBS WebSocket Plug-in¶

Now Playing has the capability to use the OBS WebSocket Plug-in to send results directly to OBS Studio.

Installation¶

  1. Install and configure the OBS WebSocket Plug-in, keeping track of the port and password.

  2. Configure OBS to have a Text source, keeping track of the name of the source name.

  3. Open Settings from the Now Playing icon

  4. Select OBS WebSocket from the list of available input sources.

  1. Check Enable

  2. Select which type of Text source was configured. Generally, GDI+ is only available on Windows.

  3. Provide the server and port of your OBS installation and the name of the Text Source.

  4. Click Save
